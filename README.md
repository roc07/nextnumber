# Getting Started

### Spring Application Requirements
Apache Maven 3.0.0+

* [Official Apache Maven documentation](https://maven.apache.org/guides/index.html)
* [Spring Boot Maven Plugin Reference Guide](https://docs.spring.io/spring-boot/docs/2.7.8/maven-plugin/reference/html/)

### How to test automatically
* There are 15 tests provided for the controller and the service, that can be run directly

### How to test manually
* Start the application
* Call the endpoints manually by: 
  * Providing either a list of numbers or both numbers and probability weights
  * Call singleNumber or multipleNumbers GETs
#### OR 
* Use the provided postman collection:
  * Import the requests in your postman
  * Use the 4 provided requests
